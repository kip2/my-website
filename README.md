## What's this?  

自分のホームとなるwebsite。    
作ったものや好きなものについて虫干しするためのサイト。  
自分が作ってきたもの、好きなプログラミング言語、好きな漫画やゲーム、影響を受けてきたものについて語り尽くすサイトです。  

「これだけみたら自分がどういう人間か伝わるかなー」という、コミュ障なりの頑張りの結晶。  
身を結んだかは定かでない。
